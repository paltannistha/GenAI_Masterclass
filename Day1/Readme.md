# 🌟 Day 1: Introduction to GenAI & Agentic AI

## 1. What is AI?
- Artificial Intelligence = machines mimicking human intelligence  
- Examples: chatbots, recommendations, self-driving systems  

---

## 2. What is Generative AI (GenAI)?
- AI that **creates new content**
- Types of content:
  - Text (ChatGPT)
  - Images (DALL·E)
  - Code (Copilot)

👉 Key Idea: *Not just analyzing → but generating*

---

## 3. How GenAI Works (High Level)
- Based on **Large Language Models (LLMs)**
- Trained on huge data
- Predicts next word/token

👉 Simple example:
> "I am going to the ___" → "market"

---

## 4. What is Agentic AI?
- AI that can **act independently**
- Takes decisions + performs tasks
- Uses tools, memory, reasoning

👉 Example:
- Booking tickets  
- Sending emails automatically  
- Multi-step problem solving  

---

## 5. GenAI vs Agentic AI

| Feature | GenAI | Agentic AI |
|--------|------|-----------|
| Purpose | Generate content | Take actions |
| Control | User-driven | Semi-autonomous |
| Example | ChatGPT response | AI booking a trip |

---

## 6. Key Concepts to Know
- Prompt  
- Tokens  
- LLM  
- Context  
- Memory (in agents)  
- Tools (APIs, search)

---

## 7. Real-Life Use Cases
- Content creation  
- Coding assistant  
- Customer support bots  
- Personal AI assistants  

---

## 8. Limitations (Important!)
- Can hallucinate (wrong answers)
- No real understanding
- Depends on training data
- Needs human supervision

---

## 9. Future Scope
- AI moving from **assistant → autonomous agents**
- More integration in daily life & jobs

# 🤖 GenAI Chatbot (Groq)

## 📌 Overview
Simple chatbot using Groq API to generate AI responses.

---

## ⚙️ Setup
```bash
pip install groq
```
---

## Add your API key in code:
```bash
client = Groq(api_key="YOUR_API_KEY")
```
---

## Run
```bash
python app.py
```
---

## 💬 Features
- Chat with AI
- Remembers conversation
- Fast responses
